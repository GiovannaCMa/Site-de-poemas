# 📝 Site de Poemas

Um projeto desenvolvido para compartilhar poemas autorais em um ambiente digital simples, sensível e acolhedor.

## 📌 Sobre o projeto

Este site foi criado com o objetivo de publicar e organizar meus poemas, transformando a escrita em uma experiência visual na web. A proposta é proporcionar uma leitura leve, onde o design e o conteúdo se complementam.

## 🌐 Acesse o projeto

👉 [Clique aqui para visualizar o site](https://gigipoemas.netlify.app/)

## 💻 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Funcionalidades

* Exibição de poemas autorais
* Layout simples e intuitivo
* Navegação leve
* Design focado na leitura


## ✨ Objetivo

Este projeto foi desenvolvido como parte dos meus estudos, com foco em praticar desenvolvimento web e também expressar minha escrita de forma criativa.
